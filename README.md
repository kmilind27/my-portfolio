# Kumar Milind — Personal Portfolio

A cinematic, immersive 3D portfolio built with React, Three.js, GSAP, and Lenis. Features smooth scroll animations, interactive project cards, a terminal-style contact section, light/dark mode, and a live WebGL background with bloom postprocessing.

🔗 **Live:** [kmilind.netlify.app](https://kmilind.netlify.app)

---

## Tech Stack

- **React 19** + **Vite**
- **Three.js** + **React Three Fiber** + **Drei**
- **GSAP** + **ScrollTrigger** — scroll animations & title reveals
- **Lenis** — smooth scroll
- **Lucide React** — icons
- **@react-three/postprocessing** — bloom effect
- **CSS Modules** — scoped styling

## Features

- 3D animated background with floating orbs, particle field, torus rings, and moving point lights
- Bloom postprocessing for cinematic glow
- Lenis smooth scroll
- GSAP scroll-triggered animations per card/element
- 3D tilt effect on skill & project cards with dynamic glow
- Magnetic buttons on hero CTAs
- Animated typing terminal in contact section
- Staggered section title reveals with clip-path wipe
- Cursor glow trail
- Film grain overlay
- Light / Dark mode with localStorage persistence
- Fully responsive

## Projects Showcased
